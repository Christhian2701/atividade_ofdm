# atividade_ofdm
Atividade avaliativa de Comunicações Digitais 2 sobre OFDM

Para execução desse programa tenha o software octave instalado

Baixe o arquivo atividade_ofdm.m

Execute o arquivo através da interface gráfica do octave

ao executar o código ele pergunta a definição de dois parâmetros

se vai usar um número de prefixos cíclicos igual a largura do canal e qual a SNR (dB) que será usada

após isso a simulação é execudata, criando e salvando os gráficos em um diretório de acordo com os parâmetros definidos
